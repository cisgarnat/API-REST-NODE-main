# API REST de Gestión de Usuarios

Este proyecto es una API REST sencilla construida con Node.js y Express (v5). Proporciona operaciones CRUD (Crear, Leer, Actualizar, Eliminar) básicas para gestionar una lista de usuarios almacenada en memoria.

## Requisitos Previos

Para ejecutar este proyecto, necesitas tener instalado en tu sistema:

* **Node.js** (versión 18 o superior recomendada).
* **npm** (normalmente viene instalado con Node.js).

## Instalación

1.  Clona o descarga este repositorio.
2.  Abre una terminal en la carpeta raíz del proyecto.
3.  Instala las dependencias necesarias ejecutando
